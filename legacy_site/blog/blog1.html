Lost in Alignment : Building a Word-Matching Engine for Any Language Pair

Identifying correspondences between words in different languages: A technical deep dive into word alignment for NMT.

NLP ALIGNMENT TRANSFORMERS

INTR0DUCTI0N

Word alignment is a fundamental task in Neural Machine Translation (NMT). It involves identifying exactly which words in a source sentence correspond to which words in the translated target. While modern Transformers handle this internally through attention mechanisms, extracting explicit alignments is still crucial for many applications like terminology extraction and interactive translation.

The Pr0blem with Hidden Alignments

Traditional NMT models often keep their "understanding" of word correspondences hidden within dense attention weights. In this post, we explore how to extract these implicitly learned alignments.

SimAlign & Contextualized Embeddings

One powerful approach is using similarity in contextualized embeddings. By comparing the internal representations of words from models like mBERT or XLM-R, we can find high-confidence matches across languages without requiring any parallel data for training.
